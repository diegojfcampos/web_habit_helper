# web_habit_helper

## Run Locally

Clone the project

```bash
  git clone https://github.com/diegojfcampos/web_habit_helper.git
```

Go to the project directory

```bash
  cd web_habit_helper
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
